# survival-horror

## Controles

- **Movimento:** `WASD`
- **Ativar/Desativar Visão Noturna:** `N`
- **Zoom In/Out durante Visão Noturna:** Scroll do Mouse
- **Ativar/Desativar Lanterna:** `F`

## Desenvolvimento

Este projeto foi desenvolvido usando Unity e C#.
